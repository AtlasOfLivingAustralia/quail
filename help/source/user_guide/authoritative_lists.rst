Using Authoritative Lists to Check for Species of Interest
==============================================================

*Note: this is only currently implemented for the Australian atlas.*

The Atlas of Living Australia has a number of species lists that are updated at least every month, if not sooner.  The 
Living Atlases QGIS Plugin has all the state and territory lists, as well as migratory and non-native lists.  While what 
the lists are called are straightforward, what each list type is and why it is used will be elaborated on.
